checkChildPage()

checkChildPage(string  $type, string  $id, array  $args) : array

This function is used to check to see if the page that is requested is a child page. This is a consiquence of implementing pretty URLs. The function takes the parent page type, the ID of the parent and the arguments that have been passed to the page. The page variables are passed back in the form of an array.

Parameters

string $type

The parent page type
string $id

The ID of the parent
array $args

The array of arguments passed to the page

Returns

array —

An array with the page type, new page ID and the new page args
